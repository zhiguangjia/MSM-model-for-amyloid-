2  Analysis

   For simple Kon Koff :

      Run following three line

         ./average_result.py wt/ton_success.dat
         ./average_result.py wt/ton_fail.dat
         ./average_result.py wt/toff.dat

      each script will output two number : average time (in ns) and howmany times it occures 
            Note: for "./average_result.py wt/toff.dat", the number of occur should equal to the number of circles in step 1. This is because this data is generated by MSM taht simulatate fully-bound inregestered peptides dissociated from core surface. In these simuations, the simulaton will not stop until the peptide enter the bulk solvent.
